## 项目介绍
* Mozi是一个基于 `Qt5` 开发的跨平台桌面诗词软件，我们在学业之余开发了Mozi。
* 我们用木樨给它命名，希望能让传统文化像桂花一样，香远益清，被更广泛的传播。
## 安装/使用
* 开发环境：Qt 5.15.0 （Linux） Qt 5.14.2（Windows）
* 目前只有Win32二进制程序包，Linux暂时无法打包。:confused:（如果有会的大佬可以联系我们 :stuck_out_tongue_winking_eye:）
